# 🏠 House Price Prediction

## 📌 Overview
This project predicts house prices using Machine Learning.

## 📊 Dataset
A simple dataset with:
- Area
- Bedrooms
- Age

## ⚙️ Algorithm Used
- Linear Regression

## 🚀 Results
The model predicts house prices based on input features.

## 📈 Output
Graph showing area vs price is generated.

- pandas
- scikit-learn
- matplotlib
